My Projects

  • Tic Tac Toe

    The Tic Tac Toe web application allows users to play the classic game against either another player or an AI opponent. The interface provides a grid-based game board where players take turns placing their respective markers (X or O) in empty squares. The game dynamically updates to indicate whose turn it is and displays a message when the game is won or ends in a draw.
    Key Features:
    Grid-Based Game Board: The game board consists of a 3x3 grid where players can place their markers.
    Player vs. Player Mode: Users can choose to play against another player, taking turns placing X and O markers.
    Player vs. AI Mode: Users can choose to play against an AI opponent, with the AI making random moves as the O player.
    Dynamic Message Display: The application displays messages indicating whose turn it is, who wins, or when the game ends in a draw.
    Reset Button: Users can reset the game at any time to start a new round or switch between player modes.
    Overall, the Tic Tac Toe application provides an interactive and engaging experience for users to enjoy the classic game either against a friend or challenging an AI opponent.

  • Stopwatch

    The Stopwatch is a simple web application that allows users to measure elapsed time. It provides basic functionality such as starting, pausing, and resetting the timer. The interface displays the elapsed time in hours, minutes, seconds, and milliseconds, updating in real-time as the timer runs.
    Key Features:
    Start Button: Initiates the stopwatch timer when clicked, displaying the elapsed time since it was started.
    Pause Button: Halts the stopwatch timer when clicked, allowing users to resume from where they left off.
    Reset Button: Resets the stopwatch timer and sets the elapsed time back to zero when clicked.
    Real-time Updates: The elapsed time is displayed in real-time, updating every 10 milliseconds to provide accurate measurements.
    Responsive Design: The interface is designed to be responsive and user-friendly, with clear buttons and a visually appealing layout.
    Overall, the Stopwatch application offers a convenient tool for users to track time intervals for various purposes, such as timing exercises, tasks, or events.
